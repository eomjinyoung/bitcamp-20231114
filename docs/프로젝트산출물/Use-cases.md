# 유스케이스 목록 예시

## 액터
- 사용자: 로그인 하지 않은 사용자
  - 회원: 회원 가입한 사용자
    - 학생: 개인 출석 및 과제를 관리하는 회원
    - 매니저: 전체 과정을 관리하는 회원
    - 강사: 개인 강의를 관리하는 회원
    - 관리자: 전체 시스템을 관리하는 회원
